Para ejecutar el procesador hay dos opciones:

Archivo por defecto src\recursos\prueba.txt.

1. La primera opcion es ejecutar el archivo run.bat que abrira automaticamente una consolo que nos pedira introducir el archivo que queremos analizar, 
o dejar en blaco para usar el archivo por defecto. 

2. En caso de que no se pueda ejecutar de la primera manera, se podra usar la consola y ejecutar el archivo con el comando >java -jar procesador.jar (nombre archivo)<,
en caso de no poner nada en el nombre archivo (sin parentesis, solo el nombre) analizara el archivo por defecto.

El resultado de la ejecucion se puede encontrar en src/recursos
